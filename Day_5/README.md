# Graph Machine Learning Course

The course material and notebooks for learning Graph Machine Learning (GML) in biological and multi-omics applications is hosted at [MoNA/course_graph_machine_learning repository](https://github.com/Multiomics-Analytics-Group/course_graph_machine_learning)

You do not need to clone the repository to follow the course content. However, if you would like to work with the files locally, use the following commands:

```bash
git clone https://github.com/Multiomics-Analytics-Group/course_graph_machine_learning.git
cd course_graph_machine_learning
```

# Repository Structure
